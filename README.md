##  Overview
This project focuses on analyzing customer churn to identify patterns, risk factors, and actionable insights. The aim is to understand **why customers leave** and to build predictive models that help reduce churn.  

## Software Used 
- SQL (analysis and cleaning)
- Python (Visualising and Machine Learning)
- Tableau (Visualising)

## Visualisations 
- Layered hsitograms comparing continuous features against wether they chruned or not
- Bar charts for discrete data to compare Churn
- Pie charts to Show Distribution 

## Modelling 
XGBoost was Created and data from Kaggle was inputted a confusion matrix was made and importance of features was plotted on a bar chart

## Results 
- Key churn drivers identified (number of products, Is active member, Age)
- The model can now predict if future customers churn or not
- Tableau dashboard shows clear insights for the data
